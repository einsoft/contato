Help: Contato

Descrição

Contatato é um módulo para Xoops 2.5.x.

Instalar/desinstalar

No special measures necessary, follow the standard installation process – extract the "MyModule" folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
